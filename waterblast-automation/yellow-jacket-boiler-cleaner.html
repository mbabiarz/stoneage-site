Yellow Jacket

Share
  • E-mail link

The StoneAge Yellow Jacket is an automated high flow water lancing system specifically designed for online boiler cleaning. The system features a pivoting lance that allows the waterjet to be precisely aimed for productive removal of excessive slag from key locations inside a boiler while the boiler is still online.

Features

  • Lance pivot angle can be set at multiple positions to provide maximum range of motion
  • Modular design with pinned connections for portability and quick assembly/disassembly
  • Stand raises/lowers and telescopes for positioning flexibility
  • Compact rigid base with multiple anchor locations
  • Lower frame has integral wheels and can be moved similar to a furniture dolly
  • Remote operation from corded pendant station
  • All functions utilize low voltage electrical power for precise positioning
  • Remote dump control available
